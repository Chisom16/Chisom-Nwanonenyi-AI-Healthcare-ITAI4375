# Midterm Project: AI in Healthcare Glossary

## Problem
This project focused on understanding key AI concepts used in healthcare.

## Approach
I created a glossary of important AI terms, including definitions and real-world healthcare examples.

## Key Concepts
- Artificial Intelligence
- Machine Learning
- Predictive Analytics
- NLP

## Results / Insights
Understanding AI terminology improves communication between healthcare and technology professionals.

## Reflection
This project helped me build a strong conceptual foundation for understanding AI in healthcare.
