# Assignment 9: Reinforcement Learning in Healthcare

## Problem
This assignment explored how reinforcement learning can be applied to healthcare decision-making.

## Approach
I designed a system that adjusts medication dosage based on patient data and outcomes.

## Key Concepts
- Reinforcement learning
- Decision-making systems
- Patient monitoring

## Results / Insights
RL can optimize treatment decisions over time but raises concerns about safety and trust.

## Reflection
This assignment showed the potential of adaptive AI systems in improving patient care.
