# Assignment 8: Natural Language Processing in Healthcare

## Problem
This assignment explored how NLP can be used to analyze unstructured medical records.

## Approach
I examined how NLP extracts key information from clinical text and evaluated its benefits and limitations.

## Key Concepts
- Natural Language Processing
- Unstructured data
- Clinical documentation

## Results / Insights
NLP can improve efficiency and support decision-making but raises concerns about accuracy and privacy.

## Reflection
This assignment helped me understand the importance of NLP in healthcare since much data is unstructured.
